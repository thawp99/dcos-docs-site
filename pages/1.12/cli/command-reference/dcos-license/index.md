```
dcos license
Usage:
  dcos license list [--output <file_path>]
  dcos license get [<id>|active] [--output <file_path>] [--decryption-key]
  dcos license audit get [<id>|active] [--output <file_path>] [--decrypt]
  dcos license status [--terms] [--breaches]
```
